# The Price of Dissent

**Rise Iran Intelligence Monitor — Protest Chronicle Dashboard**

A visual chronicle of state violence against Iranian protesters, 1988–2026.

## Deploy on Netlify

1. Push to GitHub
2. Connect repo in Netlify
3. Build command: `npm run build`
4. Publish directory: `out`

## Development

```bash
npm install
npm run dev
```
